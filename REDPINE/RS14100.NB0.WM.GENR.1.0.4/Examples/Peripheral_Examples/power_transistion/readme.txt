Functional description:
===============================================================================================
This example configures the RTC-SEC based wake up and makes the CPU power transistions from 
PS4 to PS2 then Sleep and Up on wake up again transistion to PS4 state. This repeats.
===============================================================================================
 
Assistance software:
---------------------
None

===============================================================================================
Usage Procedure:
===============================================================================================
1.Build and run the program 
2.Connect the power meter at J19 in smart EVK to see the sleep current.
