Function description:
============================================================================================================
The UART example shows how to use the UART loopback mode.
============================================================================================================
Assistance tool:
---------------------
Smart EVK

Assistance software:
---------------------
None

============================================================================================================
Usage Procedure:
============================================================================================================
TX Pin GPIO_6 :J1.7  <-> RX Pin GPIO_7 :J1.9(Loop back)
It will sends and receives data in loopback mode.Watch the tx_buffer and rx_buffer on watch window 
===============================================================================================
Pass criteria
===============================================================================================
Both rx_buffer and tx_buffer data should be same.
