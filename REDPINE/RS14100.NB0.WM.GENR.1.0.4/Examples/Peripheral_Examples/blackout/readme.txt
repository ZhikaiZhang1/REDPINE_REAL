Function description:
===============================================================================================
The blackout example will disable blackout reset in deep sleep .
===============================================================================================
Environment
===============================================================================================
Test Board:
---------------------
Smart EVK 

Assistance software:
---------------------
None

===============================================================================================
Usage Procedure:
===============================================================================================
compile and run the application.

===============================================================================================
Pass criteria:
===============================================================================================
connect the logic analyzer to reset pin and if the volltage is the less than the reset voltage 
then the reset pin should fall to low.
===============================================================================================
===============================================================================================
Note:In active state even if disable the blackout it wont disable.
