Functional description:
===============================================================================================
This Example configures the processor state from PS4 to PS0 sleep and then wakes up from deep sleep  based on RTC seconds interrupts with flash based execution. 
===============================================================================================
 
Assistance software:
---------------------
None

===============================================================================================
Usage Procedure:
===============================================================================================
1.Build and run the program 
2.Connect the power meter at J19 in smart EVK to see the sleep current.
