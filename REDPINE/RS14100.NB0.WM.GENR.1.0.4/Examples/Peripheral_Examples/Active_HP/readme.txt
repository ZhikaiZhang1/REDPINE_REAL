Functional description:
===============================================================================================
This example executes the Coremark in PS4 state @ 200MHz.  Probe the current meter to the measuring point and measure the current.
===============================================================================================
 
Assistance software:
---------------------
None

===============================================================================================
Usage Procedure:
===============================================================================================
1.Build and run the program 
2.Connect the power meter at J19 in smart EVK to see the sleep current.
